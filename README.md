# java-cli-gradle-concurrent-fibonacci

## Description
A POC for a java concurrent threads to
find nth Fibonacci.
There is a threashold of 30, after this
it either taxes the cpu too much or the result
make no sense.

## Tech stack
- java
- gradle

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
